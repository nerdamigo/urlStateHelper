Nerdamigo's Url State Helper!

When you need to easily persist some level of state such that users sharing urls can easily return to the same place, you need Nerdamigo's url state helper.

Registering an object for persistence to the querystring is as simple as a call to $urlStateHelper.register($scope, 'paramName'). A property is added to $scope, and until the $scope is $destroyed, the object will persist.

Any external changes to the url (think back/forward) cause an $urlStateHelperExternalChange event to be broadcast from $rootScope
